# OpenFL samples

This is the sample corpus root for the OpenFL reference surface.

Each sample directory can eventually contain parallel columns such as:

- `flight/` for the Flight implementation
- `openfl/` for the TypeScript OpenFL implementation
- `openfl-haxe/` for the source-of-truth Haxe reference

The detached branch you shared already uses that shape. This repo keeps the path stable so the harness and corpus can converge without another layout change.
